# Unsigned-binary-multiplication

Unsigned-binary-multiplication-algorithm with C

컴퓨터가 수행하는 부호없는 이진 정수를 곱셈하는 알고리즘을 구현해보았습니다. 

캐리와 쉬프터 연산을 시각적으로 볼 수 있게 만들어 보았습니다.
